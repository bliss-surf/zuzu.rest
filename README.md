# zuzu.rest

minimal file upload service. drop files, get links.

## what

- upload files via web interface
- get short shareable links
- images display instantly
- proxies through [yuu.pm](https://github.com/uIvPuGpT/file-host)

## run

```bash
npm install
npm run dev
```

open [http://localhost:3000](http://localhost:3000)

## deploy

built with next.js. deploy to vercel or any node host.
